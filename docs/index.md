## 安装

```bash
# npm
npm install zhiman-design

# yarn
yarn add zhiman-design

# pnpm
pnpm add zhiman-design
```

## 快速上手

```bash
import React from 'react';
import { Button } from 'zhiman-design';

const App = () => (
  <Button type="primary">Hello zhiman-design!</Button>
);

export default App;
```
